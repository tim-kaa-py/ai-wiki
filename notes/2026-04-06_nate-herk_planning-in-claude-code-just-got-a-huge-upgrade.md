# Ingest Notes

**Source:** [Planning In Claude Code Just Got a Huge Upgrade](https://www.youtube.com/watch?v=T4fXb3sbJIo)

## User Focus
- The comparison between Ultra Plan and regular plan mode in Claude Code

## Confirmed Discoveries
A. [03:29–03:58] — Ultra plan is CLI only; desktop app and VS Code extension silently do nothing when you type `ultra plan`.
B. [10:26–11:01] — Requires a Git/GitHub-synced repo; local-only projects get rejected.
C. [09:26–10:25] — API billing users can't use ultra plan (Pro/Max subscription required); one run consumed ~1% of a Max 20x allotment.
D. [11:01–12:30] — Custom skills aren't auto-invoked by the cloud planner; must name them explicitly in a review comment.
E. [13:59–14:23] — Terminal stays unblocked while ultra plan runs in the cloud (vs blocked in local plan mode).
