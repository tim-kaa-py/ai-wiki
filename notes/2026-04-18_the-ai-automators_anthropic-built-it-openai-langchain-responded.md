## Confirmed Discoveries
- A. [05:22-06:42] Anthropic Managed Agents vs AWS Bedrock AgentCore & Google Vertex AI Agent Builder — rivals are model-agnostic; "brain vs hands" separation isn't new.
- C. [04:16-05:07] The "meta harness" framing — Anthropic argues outsourcing the harness is a feature because harnesses must evolve with models.
- D. [19:03-20:18] Products that ARE agents (outside the build-to-buy spectrum) — Claude Code, OpenClaude — distinction between building agents vs. using agent products.
- E. [02:58-03:27] What Claude Managed Agents promised but didn't deliver — outcome-based tasks, multi-agent orchestration, stateful memory, evaluator agent all locked behind research preview form.
- F. [20:18-21:02] Decision heuristics for choosing a tier — full control → tier 1-2; fast time to market → tier 3-5; model flexibility → model-agnostic; compliance → self-hosted.
